Examples

This section shows how to use the plugin.

Download a File

A file can be downloaded by using the following code. The file is streamed to disk directly as data chunks are received and uses very little memory.

Blueprints C++ 
// Includes the plugin's header.
#include "FileDownloaderPro.h"
// Starts the download of the file.
FileDownloaderPro::DownloadAsFile(
    // URL of the file.
    TEXT("https://mysite/myfile.zip"),
    // Where to save the file.
    FPaths::ProjectDir() / TEXT("myfile.zip"),
    // Callback called when the file is downloaded.
    FDownloadFileDelegate::CreateLambda(
        [](FFileDownloaderProError Error)
    {
        if (Error)
        {
            // Something went wrong.
            UE_LOG(LogTemp, Error, TEXT("Failed to download file: %s"), *Error.Message);
        }
        else
        {
            // File downloaded.
        }
    })
);

Download in Memory

A file can be downloaded in memory using the following code. The full file is returned and stored in memory.

Blueprints C++ 
// Includes the plugin's header.
#include "FileDownloaderPro.h"
// Starts the download of the file.
FileDownloaderPro::DownloadAsBytes(
    // URL of the file.
    TEXT("https://mysite/myfile.zip"),
    // Callback called when the file is downloaded.
    FDownload64Delegate::CreateLambda(
        [](TArray64<uint8> Data, FFileDownloaderProError Error)
{
    if (Error)
    {
        // Something went wrong.
        UE_LOG(LogTemp, Error, TEXT("Failed to download file: %s"), *Error.Message);
    }
    else
    {
        // File downloaded, `Data` contains the file as binary data in memory.
    }
}));

Download as Chunks

A file can be downloaded as chunks. A callback fires each time a chunk of data is received and we never store the full file in memory.

Blueprints C++ 
// Includes the plugin's header.
#include "FileDownloaderPro.h"
// Starts the download of the file.
FileDownloaderPro::DownloadAsBytesChunked(
    // URL of the file.
    TEXT("https://mysite/myfile.zip"),
    FDownloadChunkDelegate::CreateLambda(
        [](void* Data, SIZE_T Size)
    {
        // New chunk received of size `Size`
    }),
    // Callback called when the file is downloaded.
    FDownloadFileDelegate::CreateLambda(
        [](FFileDownloaderProError Error)
    {
        if (Error)
        {
            // Something went wrong.
            UE_LOG(LogTemp, Error, TEXT("Failed to download file: %s"), *Error.Message);
        }
        else
        {
            // File downloaded,.
        }
    })
);

    {
        // File downloaded, `Texture` contains the image.

Cancel a Download

A download can be cancelled using the Controller. Cancelling a download also fires the On Error callback if the download did not complete before.

Blueprints C++ 
// Includes the plugin's header.
#include "FileDownloaderPro.h"
// Starts the download of the file.
FDownloadController Controller = FileDownloaderPro::DownloadAsFile(
    // URL of the file.
    TEXT("https://mysite/myfile.zip"),
    // Where to save the file.
    FPaths::ProjectDir() / TEXT("myfile.zip"),
    // Callback called when the file is downloaded.
    FDownloadFileDelegate::CreateLambda(
        [](FFileDownloaderProError Error)
    {
        if (Error)
        {
            // Something went wrong.
            UE_LOG(LogTemp, Error, TEXT("Failed to download file: %s"), *Error.Message);
        }
        else
        {
            // File downloaded.
        }
    })
);
// While the download is still going, we want to cancel it.
Controller.Cancel();

Show Download Progress

We can track the download progress by using the On Progress callback. The following code will print

0 Bytes/1.2GB 0%
...
743.23KB/1.2GB 0.1%
...
153.92MB/1.2GB 12.52%
...
1.2GB/1.2GB 100%

The server must return the Content-Length header to know the total size.

Blueprints C++ 
// Includes the plugin's header.
#include "FileDownloaderPro.h"
// Starts the download of the file.
FileDownloaderPro::DownloadAsFile(
    // URL of the file.
    TEXT("https://mysite/myfile.zip"),
    // Where to save the file.
    FPaths::ProjectDir() / TEXT("myfile.zip"),
    // Callback called when the file is downloaded.
    FDownloadFileDelegate::CreateLambda([](FFileDownloaderProError Error)
    {
        if (Error)
        {
            // Something went wrong.
            UE_LOG(LogTemp, Error, TEXT("Failed to download file: %s"), *Error.Message);
        }
        else
        {
            // File downloaded.
        }
    }),
    // Callback called when the download made some progress.
    FDownloadFileProgressDelegate::CreateLambda([](uint64 Downloaded, uint64 ContentLength)
    {
        UE_LOG(LogTemp, Log, TEXT("Downloaded: %ull/%ull %.2f"), Downloaded, ContentLength, static_cast<float>(Downloaded) / ContentLength * 100.f);
    })
);

Configure the HTTP Request

The HTTP request of the download can be configured using the Options parameter.

Blueprints C++ 
// Includes the plugin's header.
#include "FileDownloaderPro.h"
// Creates our custom settings.
FDownloadOptions Options;
Options.Headers.Add(TEXT("Authorization"), TEXT("Bearer ..."));
Options.Options.Add(TEXT("version"), TEXT("2"));
// Starts the download of the file.
FileDownloaderPro::DownloadAsTexture(
    // URL of the file.
    TEXT("https://mysite/myfile.zip"),
    // The custom options.
    MoveTemp(Options),
    // Callback called when the file is downloaded.
    FDownloadTextureDelegate::CreateLambda([](UTexture2D* Texture, FFileDownloaderProError Error)
    {
        if (Error)
        {
            // Something went wrong.
            UE_LOG(LogTemp, Error, TEXT("Failed to download file: %s"), *Error.Message);
        }
        else
        {
            // File downloaded, `Texture` contains the image.
        }
    })
);
